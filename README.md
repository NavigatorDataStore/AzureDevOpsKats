# AzureDevOpsKats

## Instalation & Run Instructions

```
git clone https://github.com/stuartshay/AzureDevOpsKats.git

cd AzureDevOpsKats
dotnet restore

cd src\AzureDevOpsKats.Web\
dotnet run
```

## Web Site

The Site can be accesed at the following url

```
http://localhost:5000/
```
